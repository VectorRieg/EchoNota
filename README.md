# EchoNota
---
EchoNota é um sistema web desenvolvido com Spring Boot, JPA e MySQL, voltado para a gestão de notas escolares, com funcionalidades de cadastro de alunos, professores, e boletins.

# Funcionalidades
---
- Java
- SpringBoot
- Spring Data JPA
- Spring Security
- MySQL
- HTML
- CSS
- Crud
- Maven
